# Message Formatter (Java)

## 📌 Description
A simple console-based program that takes user input and displays the message in its original, uppercase, and lowercase formats. This project focuses on method usage and string manipulation in Java.

## 🚀 Features
- Displays the original message
- Converts message to uppercase
- Converts message to lowercase
- Uses separate methods for each operation

## 🛠️ Built With
- Java
- Scanner (user input)
- String methods (`toUpperCase()`, `toLowerCase()`)

## ▶️ How to Run
1. Clone the repository
2. Open in any Java IDE (IntelliJ, Eclipse, etc.)
3. Run `Main.java`
4. Enter a message in the console

## 📚 What I Learned
- Creating and calling methods
- Passing arguments between methods
- Working with built-in string methods
- Structuring code for readability and reuse
